# Dotfiles

## ⚠️ Won't be update now i use MacOS

A curated collection of my personal configuration files for the tools and applications I use every day. These dotfiles help me keep a consistent, efficient setup across machines and make it easy to version-control and iterate on my workflow.

## What’s Inside

* Configuration for shells, editors, terminal tools, and development utilities
* Customizations focused on clarity, speed, and repeatability
* Scripts and helpers to streamline setup on new systems

## Usage

You can clone this repository and symlink the files to your home directory, or adapt individual configs as needed. Feel free to explore, borrow, or modify anything that fits your workflow.
